# 🛰️ Multimodal Satellite Property Valuation

## 📌 Overview
This project predicts property prices using:
- 🖼️ Satellite images
- 📊 Tabular features

## 🚀 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Pandas / NumPy

## 🧠 Model
- CNN for image features
- Dense NN for tabular data
- Combined multimodal architecture

## 📊 Results
- MAE: ~₹109930

## 📁 Project Structure
- `notebooks/` → experimentation
- `scripts/` → data processing
- `main.py` → training pipeline

## ⚠️ Note
Raw images are not included due to size constraints.
